Une collection de projets communautaires construits sur HopCoderX.

:::note
Vous souhaitez ajouter votre projet lié à HopCoderX à cette liste ? Soumettez un PR.
:::

Vous pouvez également consulter [awesome-HopCoderX](https://github.com/awesome-HopCoderX/awesome-HopCoderX) et [HopCoderX.cafe](https://HopCoderX.cafe), une communauté qui regroupe l'écosystème HopCoderX.
