ชุดโครงการชุมชนที่สร้างขึ้นบน HopCoderX

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ HopCoderX ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-HopCoderX](https://github.com/awesome-HopCoderX/awesome-HopCoderX) และ [HopCoderX.cafe](https://HopCoderX.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
