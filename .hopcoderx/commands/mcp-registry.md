Browse and install MCP servers from the built-in registry.

The registry includes pre-configured MCP servers for:
- Adobe Creative Suite (After Effects, Photoshop, Illustrator, InDesign, XD)
- Cloud services (Railway, Shopify)
- Automation tools (n8n)

Usage: @mcp-registry

Categories:
- Adobe Creative Suite: Control Adobe applications
- Cloud Services: Deploy and manage cloud resources
- Automation: Workflow automation and orchestration
- Design: Design and prototyping tools
- Development: Developer tools and utilities
- Productivity: Productivity and task management

Installation:
1. Use @mcp-registry to browse available servers
2. Or install directly with: HopCoderX mcp install <name>

Platform Compatibility:
- Servers are filtered by your current platform
- Some servers are platform-specific (e.g., Photoshop = Windows only)
- Cross-platform servers work on all supported platforms

See also:
- HopCoderX mcp install <name> - Install a specific server
- HopCoderX mcp setup adobe-suite - Install all Adobe MCPs
- @mcp-adobe-setup - Detailed Adobe setup instructions
